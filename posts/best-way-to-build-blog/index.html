终极构建博客网站方案

Monday, May 4, 2020

最终效果

创建 md 文件，运行 ./deploy.sh 生产静态文件 && 自动推送到 github，出发 vercel 的 webhook，自动部署到 vercel 的生产环境

套餐组合

  • 云服务 - vercel
  • 博客框架 - hugo
  • 代码存放 - github
  • 图床 - github
  • 上传图片工具 - upic
